# Map Generator #procjam 2020


## Install dependencies

```bash
$ npm install
```


## Build project

Before first build:
```bash
$ mkdir dist
```
(Re)build:
```bash
$ ./build.sh
```


## Run the app

Just open `./dist/index.html` in a browser
