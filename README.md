# Large World Generator #procjam 2020


## Install dependencies

```bash
$ npm install
```


## Build project

```bash
$ ./build.sh
```


## Run development web server

Listening on http://127.0.0.1:3000/ to serve the contents of the `dist` folder

```bash
$ node httpd
```
